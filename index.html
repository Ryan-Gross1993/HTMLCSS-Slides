What is a website made of?

  • The Structure Layer (HTML)
  • The Presentation Layer (CSS)

What is HTML?

HyperText Markup Language!

Originated in the early 90s.

Current version is HTML5 ('08)

HTML Structure: Elements!

An element is an individual HTML component


    Can either be:
  • Container Element: Can contain other elements
    Example: Paragraph element containing text
    <p> A paragraph </p>

  • Stand-alone element:
    Example: Images and line breaks
    <img src="/..."/img>

More HTML Structure!

  • Tags: Marks the beginning and end of an element
  • Attributes: Additional info about the element
    Such as class, id, and source
  • Values: The value assigned to an attribute

About Tags

Website content must be between HTML tags
All elements are nested inside each other.

  • Paragraphs in a list
  • A link inside a header
  • HTML elements inside a page body

Doctype

First thing on an HTML page

It tells us what markup language the class should use

<!DOCTYPE html>

Head, Title, and Body Tags

  • Head: Where the Title Tag goes
  • Title: See the top of the site?
  • Body: Where your content goes

Images and Links

Links

  • The HyperText Reference (Href) specifies the link!

Divs w/ Classes and IDs

Divs: Sections of content in an HTML page

Can group elements together w/ IDs and Classes

Can use these IDs and Classes for CSS styling!

  • Classes: Used for multiple groups of elements
    Multiple lists, but with the same font color.

  • IDs: Use for formatting a specific group

What is CSS?

    Cascading Style Sheets
  • Allows you to modify HTML elements
  • Colors, Borders, Size, and Positioning

Colors in CSS

    You can change the colors of:
  • Text: color: red
  • Background: background-color: #000000;
  • Useful Link: htmlcolorcodes.com

Borders

Edges around elements like images
Borders are determined by:
thickness (px), style and color
Ex: A solid black border
border: 1px solid black;

Ex: A dotted red border
border: 3px dotted #FF5733;

Sizing Text in CSS

Done in units of em; relative unit to the font.


Better than using px., as that can vary on different screens


font-size: 1.4em;
You can make text, images, and other elements bigger/smaller

Margins and Padding

Margins: Transparent box that separates elements Padding: Space b/w border and the element
padding-left: 10px;
padding-right: 10px;
padding-top: 10px;
padding-bottom: 10px;
OR
padding: 10px 10px 10px 10px
(top, right, bottom, left)
margin-left: 10px;
margin-right: 10px;
margin-top: 10px;
margin-bottom: 10px;
OR
margin: 10px 10px 10px 10px
(top, right, bottom, left)
